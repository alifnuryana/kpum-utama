## E-Voting | KPUM UTAMA

KPUM UTAMA adalah sebuah website e-voting yang dilakukan di lingkungan kampus Universitas Widyatama.

Untuk melakukan pemilihan :
- MPM
- Presma
- Senat
- UKM

## Requirements

- PHP 7.3+
- Node 14+
- MySql 5.7+
